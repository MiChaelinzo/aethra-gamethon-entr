## Recording Tips

### Setup

1. **Browser**: Use Chrome/Edge in fullscreen mode (F11)
2. **Resolution**: Set window to exactly 1920x1080
3. **Mouse Recording**: Enable cursor in recording software
4. **Audio**: 
   - Record system audio (game music + sound effects)
   - Optional: Add voiceover narration later
   - Keep music volume at 60-70% for clarity

### Software Options

**Free**:
- **OBS Studio** (open source, powerful)
  - Download: https://obsproject.com
  - Tutorial: https://www.youtube.com/watch?v=EuSUPpoi0Vs
  
**Browser-Based**:
- Chrome/Edge built-in screen recording
- Loom (free tier: https://loom.com)

**Paid** (if you have access):
- Camtasia ($299)
- ScreenFlow (Mac, $169)
- Adobe Premiere Pro (subscription)

### Recording Settings (OBS Studio)

1. **Add Display Capture** source
2. **Settings > Video**:
   - Base Resolution: 1920x1080
   - Output Resolution: 1920x1080
   - FPS: 60 (or 30 if file size is an issue)
3. **Settings > Output**:
   - Encoder: x264
   - Rate Control: CRF
   - CRF: 18 (higher quality) to 23 (smaller file)
4. **Record** button starts recording

### Rehearse First!

- Do 2-3 practice runs before final recording
- Write down your exact clicks/actions
- Time each section to stay on schedule
- Test audio levels (music not too loud)

### During Recording

- **Move slowly**: Don't rush clicks
- **Pause briefly**: Give viewers time to see each screen
- **Be smooth**: Avoid jerky mouse movements
- **Show success**: Let animations complete, show celebrations
- **Highlight features**: Make sure unique features are obvious

### Post-Recording

1. **Trim**: Remove any dead time at start/end
2. **Add text overlays**: 
   - Use video editor to add feature callouts
   - Subtitle any narration
   - Highlight key numbers (scores, combos)
3. **Export**:
   - Format: MP4 (H.264)
   - Resolution: 1920x1080
   - Bitrate: 5-10 Mbps
   - Audio: AAC, 192 kbps
4. **Review**: Watch entire video before uploading
